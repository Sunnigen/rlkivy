# rlkivy

rlkivy is a RogueLike Game made in Python using Kivy. The purpose is to make a Kivy version of the [roguelike tutorial](http://rogueliketutorials.com/)

## Installation

Python 3.7

```bash
pip install -r requirements.txt
```
